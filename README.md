## 왜 만들었나요?

Claude Code 터미널 여러 개 띄워놓고 **"이 세션 뭐하고 있었지?"** 했던 적 있으신가요?

저는 항상 **4~6개의 세션**을 동시에 켜두고 병렬 작업을 하고 있었어요.

- 어떤 세션이 아직 작업 중인지, 끝났는지 확인하려면 **터미널을 하나하나 들어가봐야** 했고
- 확인이 늦어서 **시간이 낭비**되기도 하고
- 어떤 세션에서 뭘 하고 있었는지 **기억이 안 날 때**도 많았어요

그래서 직접 만들었습니다.

> Claude Code 멀티 세션 실시간 대시보드, **claude-paws** 🐾

paws는 동물들의 발자국이라는 뜻이에요.
Claude Code 세션들의 발자취를 따라가며, 지금 어디서 뭘 하고 있는지 한눈에 볼 수 있도록 만들었어요.
12종의 픽셀 동물 마스코트가 각 세션을 대신해서 걸어다닙니다 🐱🐧

<br />

## 뭘 할 수 있나요?

`paws` 한 줄이면, 실행 중인 **모든 세션을 한 화면에서** 볼 수 있어요.

- 각 세션의 상태를 **실시간**으로 확인
- 클릭하면 해당 **터미널로 바로 이동**
- 작업이 끝나면 **macOS 알림**과 **Toast 알림**으로 알려줘요

놓칠 걱정 없이 다른 작업에 집중할 수 있어요.

<br />

## 주요 기능

| 기능 | 설명 |
|------|------|
| **실시간 모니터링** | 여러 프로젝트의 세션 상태를 한 화면에서 확인 |
| **터미널 바로가기** | 카드에서 클릭 한 번으로 해당 터미널로 전환 |
| **세션 관리** | 모달에서 상세 정보 확인, 필요 없는 세션은 바로 삭제 |
| **알림** | macOS 알림 + Toast 알림, 대시보드에서 on/off 가능 |
| **12종 픽셀 마스코트** | 고양이, 강아지, 토끼, 곰, 펭귄, 호랑이, 사자, 알파카, 여우, 햄스터, 부엉이, 판다 |
| **걸어다니는 펫** | 마스코트가 대시보드 위를 돌아다니며 말풍선을 띄워요 |
| **다크 모드** | `D` 키 하나로 전환 |
| **드래그 & 리사이즈** | 카드 배치를 자유롭게 커스텀 |
| **팀 에이전트 시각화** | 에이전트 팀 구조를 모달에서 확인 |
| **트랜스크립트** | 카드 클릭하면 세션 타임라인 확인 |
| **키보드 단축키** | `D` 다크 모드, `E` 편집 모드, `1-5` 컬럼 수, `Esc` 모달 닫기 |
| **캐릭터 선택** | 나를 대표하는 마스코트를 골라요 |

<br />

## 어떻게 동작하나요?

- Claude Code의 **Hook 시스템**으로 세션 데이터를 자동 수집
- 외부 의존성 없이 **Node.js 내장 모듈만** 사용
- 모든 데이터는 **로컬에서만** 처리돼요

<br />

## 설치

```bash
npm install -g claude-paws
```

Node.js 18 이상이면 돼요.

<br />

## 사용법

```bash
# 대시보드 시작
paws

# hook 설치 (npm install 하면 자동으로 돼요)
paws setup

# 설치 상태 확인
paws status

# 포트 바꾸기
paws --port 3300

# 브라우저 자동으로 안 열기
paws --no-open
```

http://localhost:3200 으로 접속하면 돼요.

<br />

## 삭제

```bash
npm uninstall -g claude-paws
```

hook까지 깔끔하게 지우려면:

```bash
rm ~/.claude/hooks/session-tracker.sh
rm -rf ~/.claude/dashboard/
```

`~/.claude/settings.json`에서 `session-tracker.sh` 관련 항목도 지워주면 끝이에요.
